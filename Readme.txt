A futtatáshoz meg kell adni a jar fájlokat.
Én Eclipse-et használtam. 
Ott jobb klikk a projektre -> Build Path -> Configure Build Path... -> Libraries -> Add External JARs... és kiválasztani a két mellékelt jar fájlt.
Ezen kívül ugyanitt (a hozzáadás után) glpk-java.jar melletti nyílra rákattintani -> Native library location és megadni a w64 mappát.
